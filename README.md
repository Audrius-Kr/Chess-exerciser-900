# Chess-exerciser-900

A memory-training chess game where you play against a Stockfish bot while the board periodically blacks out, forcing you to remember the position.

## Stack

- **Backend:** ASP.NET Core 7 (C#), Entity Framework Core, SQL Server, JWT auth, Stockfish engine
- **Frontend:** React 18 (Create React App), React Router, React-Bootstrap, axios
- **Tests:** xUnit, Moq, FluentAssertions
- **Hosting:** Azure (Web App + Static Web App)
